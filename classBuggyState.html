Detailed Description

The Buggy State class is an abstract class that represents the abstract state the buggy can be in. From here all Buggy States are inherited e.g. IdleState or CalibrationState. On state transition the different methods are called. First the exit method when the state is exited and then the enter method when the state is entered. The update method is called on every update, meaning as often as possible.
