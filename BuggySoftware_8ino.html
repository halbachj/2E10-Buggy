◆ loop()

void loop ( )

Main loop of the arduino. Called as often ass possible.

This function is used to update all members of the buggy as often as possible. This method will call the Buggy update method which intern updates the necessary components of the buggy dependent on the state. The function also keeps track of execution time in ms, which is passed onto the subsequent update functions if they need it.
