Detailed Description

The line follower class is a helper class that allows for easy control of the buggy.

It holds both motors ans sensors and the PID controller for easy access. On update the line follower class will poll the IR sensors calculate the error and turn based on the calculated error. It also has PID capabilites for future PID based turning.
